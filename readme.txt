A dashboard toolkit with advanced interaction controls.

== Description ==

Use the Submissions Dashboard plugin to view and manage your newly created **submissions** table. 

Add additional columns through your preferred database admin tool (e.g., phpmyadmin). Columns will dynamically update on the existing **submissions** table. 

== Features ==
* Simple & Intuitive
* Powerful Interaction Tools
* Filter By Keywords
* Filter By Date Range
* Export Data In Different Formats

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Click** on "Upload Plugin" and upload the plugin folder
3. **Activate** Submissions Dashboard from your Plugins page
4. **Click** on the new menu item "Submissions" and view the newly created "submissions" table.
5. **Add** any additional columns through your preferred database administration tool (e.g., phpmyadmin)
